資料設定

設定業主、匯入 Meta CSV、設定欄位與分組、儲存為月份報表

0. 業主資訊 業主名稱與 Logo 會套用到所有頁面、瀏覽器分頁圖示與業主版

無 Logo
業主 Logo（取代瀏覽器分頁圖示）
建議用方形圖檔（PNG/JPG/SVG），會自動裁成 64×64 顯示

1. 匯入 CSV 秦小姐專案僅使用 TWD 帳號

2. 欄位對應 通常自動偵測｜「成果」欄會依分組自動轉成新追蹤者或點擊次數

3. 活動分組 關鍵字命中即歸入該組

3b. 受眾名稱重新命名 在「每月成效」表格按受眾右側 ✏️ 即可重新命名；此處可檢視／移除

4. 素材圖批次上傳 檔名（去副檔名、忽略空白／底線／橫線）會自動 match 到對應的廣告名稱

6. 自動同步到 GitHub 設定一次 Token，之後儲存月份時自動 push JSON，業主端自動看到最新

🔐 如何取得 Token：
  1. 到 https://github.com/settings/tokens?type=beta（Fine-grained Token，更安全）
  2. 按 Generate new token → 名稱填「ads-sync」、過期 90 天
  3. Repository access 選 Only select repositories → 勾 drLiao
  4. Repository permissions：Contents: Read and write
  5. Generate → 複製 ghp_ 開頭那串貼到上方
  6. ⚠️ Token 等於密碼，不要外流！只存在你瀏覽器的 localStorage

5. 月份儲存 把這次匯入的資料存成一個月份，方便日後在「成效總覽」切換比較

儲存後可在「成效總覽 / 受眾包成效 / 素材表現 / 原始資料檢核」上方用月份切換頁籤。
